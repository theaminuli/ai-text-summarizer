Add AI-powered Summary Buttons to your WordPress site with customizable layouts and styling options.

== Description ==

AI Text Summarizer is a powerful WordPress block that allows you to create customizable button groups for your content. Built following WordPress Gutenberg standards, this block provides a flexible container for organizing multiple buttons with various layout options.

**Key Features:**

* Create horizontal or vertical button layouts
* Full support for WordPress block editor styling options
* Responsive design with mobile-friendly layouts
* Customizable spacing, colors, typography, and borders
* Support for gradients and advanced styling
* Compatible with core/button blocks
* Block transformations from paragraphs and existing buttons
* Follows WordPress accessibility standards

**Perfect For:**

* Call-to-action sections
* Navigation menus
* Product showcases
* Download sections
* Contact and social media links

== Installation ==

**From WordPress Dashboard:**

1. Navigate to Plugins > Add New
2. Search for "AI Text Summarizer"
3. Click "Install Now" and then "Activate"
4. The block will be available in the block editor under the Design category

**Manual Installation:**

1. Download the plugin zip file
2. Upload the plugin files to the `/wp-content/plugins/ai-text-summarizer` directory
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Start using the AI Text Summarizer block in your posts and pages


== Frequently Asked Questions ==

= How do I add this block to my page? =

In the block editor, click the "+" button to add a new block, search for "Summarize Button" or find it under the Design category. The block will be inserted with a default button that you can customize.

= Can I customize the button styles? =

Yes! The block supports all WordPress core button styling options including colors, gradients, typography, spacing, and borders. You can customize each button individually or apply styles to the entire button group.

= How do I create vertical button layouts? =

Select the Summarize Button block and use the layout options in the block settings sidebar to change the orientation from horizontal to vertical.

= Does it work with any WordPress theme? =

Yes, the block is built using WordPress core components and follows Gutenberg standards, making it compatible with any properly coded WordPress theme.

= Can I use this with other button blocks? =

Yes, the block is designed to work as a container for core/button blocks, ensuring full compatibility with WordPress's native button functionality.
