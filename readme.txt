= What is TL;DR? =

"TL;DR" stands for "Too Long; Didn't Read" - a popular internet acronym used to provide quick summaries of lengthy text. It acknowledges that modern readers often scan content before committing to read the full article. Adding TL;DR buttons shows you respect your readers' time.

= How It Works =

1. Add the "Summarize Button" block to any post or page
2. Customize button text ("Summarize", "Get TL;DR", "Quick Summary", etc.)
3. Link buttons to your summary sections or trigger AI summarization
4. Style buttons to match your website design
5. Publish and help readers navigate your content efficiently


== Installation ==

**From WordPress Dashboard:**

1. Navigate to Plugins > Add New
2. Search for "AI Summary"
3. Click "Install Now" and then "Activate"
4. The block will be available in the block editor under the Design category

**Manual Installation:**

1. Download the plugin zip file
2. Upload the plugin files to the `/wp-content/plugins/ai-summary` directory
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Start using the AI Summary block in your posts and pages


== Frequently Asked Questions ==

= What is TL;DR and why do I need it? =

TL;DR stands for "Too Long; Didn't Read." It's an internet acronym used to provide quick summaries of lengthy content. Adding TL;DR buttons helps readers decide if your content is relevant to them, reducing bounce rates and improving user experience.

= How do I add summarize buttons to my content? =

Simply edit any post or page in the Gutenberg editor, click the "+" button, and search for "Summarize Button" or find it under the Design category. The block will be inserted with default buttons that you can customize.

= Does this plugin actually summarize content with AI? =

Currently, the plugin provides customizable buttons that you can link to summary sections. We're actively developing AI-powered automatic summarization using Chrome's Built-in AI (Gemini Nano) and other AI services. Stay tuned for updates!

= Can I customize the button appearance? =

Absolutely! You have full control over:

â€¢ Text size, font family, and weight

= How do I create vertical button layouts? =

Select your Summarize Button block, then in the block settings sidebar (right panel), look for Layout Options and change the orientation from "Horizontal" to "Vertical."

= Is this compatible with my WordPress theme? =

Yes! The plugin is built using native WordPress Gutenberg components and follows all WordPress coding standards. It works perfectly with any properly coded WordPress theme, including FSE (Full Site Editing) themes.

= Does it work on mobile devices? =

Yes, the buttons are fully responsive and automatically adapt to different screen sizes. They look great on phones, tablets, and desktops.

= Can I use multiple summarize button blocks on one page? =

Yes! You can add as many Summarize Button blocks as you need throughout your content. Each block can be styled independently.

= Is it accessible for users with disabilities? =

Yes, the plugin follows WCAG 2.1 Level AA accessibility standards with full keyboard navigation support, proper ARIA labels, and screen reader compatibility.

= Will this slow down my website? =

No. The plugin is lightweight with minimal JavaScript and CSS. It uses native WordPress components and follows performance best practices.

= Can I translate the button text? =

Yes, all button text is fully customizable and supports WordPress translation functions. The plugin is translation-ready for multi-language websites.

= How do I connect buttons to AI summarization? =

AI integration is in development. Currently, you can link buttons to summary sections on your page using anchor links (#summary), or to separate summary pages. Automatic AI summarization will be available in a future update.
